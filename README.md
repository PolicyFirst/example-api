# README

Example Text Api
	
	- uses Rails API mode to create a simple API with 'users' and 'posts'

API Definition:
	- Swagger.yaml	


Ruby version: 2.6.6

System dependencies:
	- rspec
	- JWT
	- bcrypt
	- other default rails dependencies

Initilization: 
	- bundle install
	- rails db:setup

Database: sqllite3 development and test, postgres production

How to run the test suite: bundle exec rspec
	- verifys models
	- tests requests
